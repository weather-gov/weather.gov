# Deploying

If you need to deploy in a hurry, read [tagged-releases.md](../product/tagged-releases.md).

We usually deploy using GitLab CD. You can read more about how deployments work within each environment in [environments.md](../environments.md).

Deployments are done via Terraform. You can read more about how that works in [terraform.md](./terraform.md).

Manual deployments can be done either from the GitLab Pipelines page, by clicking New Pipeline, or by using Terraform locally.
