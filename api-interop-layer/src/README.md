# API Interop Layer Source Code

This directory contains the source code for the API Interop Layer.

## Directory Structure

- `server.js`: The Fastify server entry point.
- `main.js`: The application entry point that starts the server.
- `routes/`: API route definitions and handlers.
- `util/`: Shared utility functions.
- `data/`: Data access layer and business logic.
- `lib/`: Third-party library configurations and wrappers (e.g. Redis).
